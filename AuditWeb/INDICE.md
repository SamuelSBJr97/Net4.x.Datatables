## ??? Scripts PowerShell

### 1. start-docker.ps1
**Inicia o PostgreSQL no Docker**
```powershell
.\start-docker.ps1
```
- Verifica Docker instalado
- Inicia container
- Mostra status
- Aguarda inicialização
- Exibe informações de conexão

---

### 2. install-packages.ps1
**Instala pacotes NuGet**
```powershell
.\install-packages.ps1
```
- Baixa NuGet.exe
- Restaura todos os pacotes
- Inclui Npgsql e dependências

### 3. test-system.ps1

## ?? Comandos Mais Usados

```powershell
# Iniciar tudo
.\start-docker.ps1
.\install-packages.ps1
.\test-system.ps1

# Ver logs
docker logs -f auditlog_postgres

# Parar/Reiniciar
docker-compose down
docker-compose up -d

# Acessar PostgreSQL
docker exec -it auditlog_postgres psql -U audituser -d AuditLogDB

# Contar registros
docker exec -it auditlog_postgres psql -U audituser -d AuditLogDB -c "SELECT COUNT(*) FROM audit_logs;"
```
